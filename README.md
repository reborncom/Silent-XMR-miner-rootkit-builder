# silent-XMR-miner-rootkit-builder
This is free builder of silent xmr miner

Miner features:
  - Huge pages support
  - Slow mode support
  - Work only on idle mode
  - Workable threads settings
  - Init threads settings
  - CPU priority settings
  - Custom path to modules
Rootkit features:
  - Watchdog
  - AVCheck
  - Hider
